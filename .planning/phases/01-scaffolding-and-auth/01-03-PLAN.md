  <action>
    1. Complete `laundry_advisor/settings/prod.py`:
       ```python
       from .base import *
       import dj_database_url

       DEBUG = False

       # Security
       SECRET_KEY = env('SECRET_KEY')  # MUST be set in Render env vars
       ALLOWED_HOSTS = []

       # Add Render external hostname
       RENDER_EXTERNAL_HOSTNAME = env('RENDER_EXTERNAL_HOSTNAME', default=None)
       if RENDER_EXTERNAL_HOSTNAME:
           ALLOWED_HOSTS.append(RENDER_EXTERNAL_HOSTNAME)

       # Database — PostgreSQL via DATABASE_URL
       DATABASES = {
           'default': dj_database_url.config(
               conn_max_age=600,
               conn_health_checks=True,
           )
       }

       # Static files — WhiteNoise
       MIDDLEWARE.insert(
           MIDDLEWARE.index('django.middleware.security.SecurityMiddleware') + 1,
           'whitenoise.middleware.WhiteNoiseMiddleware',
       )

       STATIC_ROOT = BASE_DIR / 'staticfiles'
       STORAGES = {
           'staticfiles': {
               'BACKEND': 'whitenoise.storage.CompressedManifestStaticFilesStorage',
           },
       }

       # Tailwind CLI — do NOT auto-download in production (binary installed during build)
       TAILWIND_CLI_AUTOMATIC_DOWNLOAD = False

       # Security settings for production
       SECURE_PROXY_SSL_HEADER = ('HTTP_X_FORWARDED_PROTO', 'https')
       SESSION_COOKIE_SECURE = True
       CSRF_COOKIE_SECURE = True
       SECURE_SSL_REDIRECT = True
       SECURE_HSTS_SECONDS = 31536000
       SECURE_HSTS_INCLUDE_SUBDOMAINS = True
       SECURE_HSTS_PRELOAD = True
       ```

    2. Ensure `laundry_advisor/wsgi.py` defaults to prod settings:
       ```python
       os.environ.setdefault('DJANGO_SETTINGS_MODULE', 'laundry_advisor.settings.prod')
       ```

    3. Create `render.yaml` — Render Blueprint:
       ```yaml
       databases:
         - name: laundryadvisor-db
           plan: free
           databaseName: laundryadvisor
           user: laundryadvisor

       services:
         - type: web
           name: laundryadvisor
           runtime: python
           plan: free
           buildCommand: "./build.sh"
           startCommand: "gunicorn laundry_advisor.wsgi:application"
           envVars:
             - key: DJANGO_SETTINGS_MODULE
               value: laundry_advisor.settings.prod
             - key: SECRET_KEY
               generateValue: true
             - key: DATABASE_URL
               fromDatabase:
                 name: laundryadvisor-db
                 property: connectionURI
             - key: RENDER_EXTERNAL_HOSTNAME
               fromService:
                 type: web
                 name: laundryadvisor
                 property: host
       ```

    4. Create `build.sh` (must be executable):
       ```bash
       #!/usr/bin/env bash
       set -o errexit

       pip install -r requirements.txt

       # Download Tailwind CLI binary and build CSS
       python manage.py tailwind download_cli
       python manage.py tailwind build

       # Collect static files (AFTER Tailwind build)
       python manage.py collectstatic --no-input

       # Run database migrations
       python manage.py migrate
       ```
       Run `chmod +x build.sh` to make it executable (important for Render).

    5. Verify `requirements.txt` has all production dependencies (should already have gunicorn, whitenoise, dj-database-url, psycopg2-binary from Plan 01-01 — confirm they're present).
