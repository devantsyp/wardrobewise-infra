## Decisions Made

- **WhiteNoise MissingFileError root cause:** `STATICFILES_DIRS = [BASE_DIR / 'assets']` causes `assets/src/main.css` (Tailwind v4 source, contains `@import "tailwindcss"`) to be collected by `collectstatic`. WhiteNoise's `CompressedManifestStaticFilesStorage` parses all CSS and tries to resolve `@import "tailwindcss"` as a static file path → looks for `src/tailwindcss` → not found → build fails. Fix: `rm -rf assets/src/` in `build.sh` after tailwind build, before collectstatic. Source CSS only needs to exist for the build step; the compiled `assets/css/app.css` is what gets served.
