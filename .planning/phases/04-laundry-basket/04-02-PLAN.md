    - `@login_required`
    - Get or select active basket: check query param `basket_id` first, else get most recently used basket for user, else None.
    - If basket found and belongs to user, update `last_used_at` by calling `basket.save()` (auto_now handles it).
    - Query analyzed garments: `Garment.objects.filter(user=request.user).annotate(has_analysis=Exists(CareAnalysis.objects.filter(garment=OuterRef('pk')))).filter(has_analysis=True).select_related('care_analysis')`
    - Query all user baskets: `Basket.objects.filter(user=request.user)` (ordered by -last_used_at via Meta)
    - Stale plan detection: if `basket` and `basket.saved_plan` and `basket.plan_saved_at`:
      ```python
      from django.db.models import Max
      last_analysis_update = CareAnalysis.objects.filter(
          garment__pk__in=basket.garment_pks,
          garment__user=request.user,
      ).aggregate(latest=Max('updated_at'))['latest']
      plan_is_stale = (last_analysis_update is not None and last_analysis_update > basket.plan_saved_at)
      ```
    - Build categories list from CATEGORY_CHOICES for the filter dropdown.
    - Context: `{'garments': analyzed_garments, 'basket': basket, 'baskets': user_baskets, 'plan_is_stale': plan_is_stale, 'categories': CATEGORY_CHOICES}`
    - Render `laundry/basket.html`

    **`plan_api(request)` — POST /basket/api/plan/:**
    - `@login_required`, `@require_POST`
    - Read `request.body` once at the top: `body = json.loads(request.body)`
    - Extract `pks = [int(pk) for pk in body.get('garment_pks', [])]`
    - Validate: if `len(pks) < 2` return `JsonResponse({'error': 'Select at least 2 garments'}, status=400)`
    - Validate: if `len(pks) > 20` return `JsonResponse({'error': 'Maximum 20 garments'}, status=400)`
    - Fetch garments: `Garment.objects.filter(pk__in=pks, user=request.user).select_related('care_analysis')`
    - Build garment dicts list:
      ```python
      garment_dicts = []
      for g in garments:
          try:
              a = g.care_analysis
          except CareAnalysis.DoesNotExist:
              continue
          garment_dicts.append({
              'pk': g.pk,
              'name': g.name,
              'photo_url': g.garment_photo.url if g.garment_photo else None,
              'category': g.get_category_display(),
              'color': g.color,
              'washing': a.washing,
              'drying': a.drying,
              'bleach': a.bleach,
              'is_delicate': a.is_delicate,
          })
      ```
    - Call `group_into_loads(garment_dicts)` and return `JsonResponse(plan)`
    - Wrap in try/except for `json.JSONDecodeError`, `ValueError`, `TypeError` returning 400.
